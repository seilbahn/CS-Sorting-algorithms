      AdvancedArrayTCreateArray(Int32, T, T, Decimal, ArrayType) Method

      This method generates an array with assigned parameters.
      The type of the returned array is the same as generic type.

      Parameters

      sizeOfArray  Int32
      The size of generating array.
      It should be within the range [1 ; 2_146_435_071].
      DON'T use int.MaxValue or System.Int32.MaxValue. There is a readonly value MAXIMUM_INDEX = 2_146_435_071 for this case.
      minimumValue  T
      Minimum value of the array elements.
      It should be within the range [type.MinValue ; type.MaxValue].
      maximumValue  T
      Maximum value of the array elements.
      It should be within the range (Minimum Value ; type.MaxValue].
      similarElementsExpectancy  Decimal  (Optional)
      This parameter is used for assigning the probability of occurrence for one value in the whole array.
      It should be within the range [0.1 ; 1.0].
      0.1 means that 10 % of the array will be filled with the same value (How many elements will be the same).
      arrayType  ArrayType  (Optional)
      It defines which type the array should be.
      Without this parameter the array will be random type.
      ArrayType.Random - the default random generated array.
      ArrayType.Sorted - the array will be sorted ascending.
      ArrayType.NearlySorted - the array will contain unsorted elements, but mostly it will be sorted ascending.
      ArrayType.Reversed - the array will be sorted descending.
      ArrayType.FewUnique - the array will contain a very limited set of values.

      Return Value

      T
      The generated array with the type of generic.
