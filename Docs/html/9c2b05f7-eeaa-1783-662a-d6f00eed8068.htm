      ArraysInfo Properties

      Properties

      InputArray The input array for printing.
      It should be a digit or char array or AdvancedArray type.
      InputArraySize The length of the input array.
      InputArrayType The type of the input array. It is created to show the character of the array.
      The arrays can be very different. Sometimes it is important to know which array type was beeing contained while calculating.
      IsInputArraySorted Is the input array ascending sorted or not.
      IsOutputArraySorted Is the output array ascending sorted or not.
      OutputArray The output array for printing.
      It should be a digit or char array or AdvancedArray type.
      OutputArraySize The length of the output array.
      OutputArrayType The type of the output array. It is created to show the character of the array.
      The arrays can be very different. Sometimes it is important to know which array type was beeing contained while calculating.
