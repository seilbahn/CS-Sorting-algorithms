      Printer Properties

      Properties

      AlgorithmsInfo The sorting algorithms information for printing.
      ArraysInfo The arrays information for printing.
      ReportsFolderName The name of the folder for reports.
      The default name is "Sorting algorithms reports".
      It changes according to the system settings.
      SortingReportsFolderPath The path to the working folder (the folder for reports). The folder is included to this path.
      WorkingFolderPath The path to the working folder (the folder for reports). The folder is not included to this path.
      The default path is the directory "My documents".
